# AI Agent Guidelines

1. **No AI Attribution**: Agents (Claude, Gemini, Copilot, Codex, etc.) must NEVER be credited as authors, contributors, or co-authors in files, code, commits, or pull requests.
2. **Authorship**: All changes must be credited to the human user directing the agent or the repository owner.
3. **Commit Messages**: Do not include AI attribution tags (e.g. `Co-authored-by: AI-agent`).
4. **Symlinks**: `CLAUDE.md` and `GEMINI.md` should be symbolic links to this file to ensure rules are read uniformly.
